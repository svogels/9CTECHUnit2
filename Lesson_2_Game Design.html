I Do: Install & Create Window

Installing Pygame

Open a terminal or command prompt and type:

pip install pygame

First Game Window Code

import pygame
import sys

# Initialize Pygame
pygame.init()

# Set up the display
WINDOW_WIDTH = 800
WINDOW_HEIGHT = 600
screen = pygame.display.set_mode((WINDOW_WIDTH, WINDOW_HEIGHT))
pygame.display.set_caption("My First Game Window")

# Define colors (R, G, B)
BLACK = (0, 0, 0)
WHITE = (255, 255, 255)

# Game loop
running = True
clock = pygame.time.Clock()

while running:
    # Handle events
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    
    # Fill the screen
    screen.fill(BLACK)
    
    # Update display
    pygame.display.flip()
    clock.tick(60)  # 60 FPS

# Quit
pygame.quit()
sys.exit()

We Do: Common Issues & Solutions

Problem: Import errors like `ModuleNotFoundError`

Solution: This means Pygame isn't installed correctly. Double-check your spelling and run pip install pygame again. Make sure you're using the correct Python interpreter in your editor.

Problem: The window is "not responding" or freezes.

Solution: Your event loop is probably missing or incorrect. Make sure the `for event in pygame.event.get():` loop is inside your main `while running:` loop.

Problem: The window opens and closes immediately.

Solution: The `while running:` loop isn't working. Check that the `running` variable is initially set to `True` and that you don't accidentally set it to `False` somewhere.

Complete Window Code Template

import pygame
import sys

# Initialize Pygame
pygame.init()

# --- CONSTANTS ---
WINDOW_WIDTH = 800
WINDOW_HEIGHT = 600
FPS = 60

# Colors (R, G, B)
BLACK = (0, 0, 0)
WHITE = (255, 255, 255)
RED = (255, 0, 0)
GREEN = (0, 255, 0)
BLUE = (0, 0, 255)

# --- SETUP ---
# Create the game window
screen = pygame.display.set_mode((WINDOW_WIDTH, WINDOW_HEIGHT))
pygame.display.set_caption("Year 9 Game Development")

# Game variables
clock = pygame.time.Clock()
running = True

# --- GAME LOOP ---
while running:
    # 1. Event Handling
    for event in pygame.event.get():
        # User clicks the 'X' button
        if event.type == pygame.QUIT:
            running = False
        # User presses a key
        elif event.type == pygame.KEYDOWN:
            # Pressing ESC also quits
            if event.key == pygame.K_ESCAPE:
                running = False
    
    # 2. Update Game State (logic goes here later)
    # (Game logic will go here)
    
    # 3. Drawing (Output)
    screen.fill(BLACK) # Clear screen
    
    # --- UPDATE DISPLAY ---
    pygame.display.flip()
    
    # Control the frame rate
    clock.tick(FPS)

# --- CLEAN UP ---
pygame.quit()
sys.exit()

Differentiation Strategies

For Students Who Need Support:

  • Provide pre-written code with comments to fill in the blanks.
  • Pair with more confident students for collaborative problem-solving.
  • Focus on completing Level 1 challenges successfully.
  • Use `print()` statements to track variables and see what the code is doing (visual debugging).

For Advanced Students:

  • Create a simple shape (e.g., `pygame.Rect`) that moves with the arrow keys.
  • Add background music using the `pygame.mixer` module.
  • Research and implement "double buffering" concepts.
  • Create a color gradient for the background instead of a solid color.
